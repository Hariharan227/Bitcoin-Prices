This code is written in PHP language and uses Apache2.4 web server.

Steps to run the page:
1) Install Apache2.4 server as guided here: https://www.sitepoint.com/how-to-install-apache-on-windows/. Follow ONLY steps 2 and 3 of this installation website.
2) Install PHP as guided here: https://www.sitepoint.com/how-to-install-php-on-windows/. Follow ONLY steps 1,2 and 4 of this installation website.
3) In the above step, download the Thread Safe PHP ZIP file.
4) Replace the C:\Apache24\conf\httpd.conf file of the Apache server folder with the httpd.conf file attached in this Bitcoins Prices repository.
5) Add the php.ini file attached in this Bitcoin Prices repository to C:\php folder.
6) Follow steps 5 and 6 of Apache installation website mentioned above.
7) Empty the C:\Apache24\htdocs folder and copy the index.php and css folder of this Bitcoin Prices repository to that htdocs folder.
8) Start the Apache2.4 server, if not already.
9) Run http://localhost/ in the browser to see the final output.

The web page output images are located in the Output Images folder of this Bitcoin Prices repository.
